# Titanic-Data-Analysis-Project_4

This project performs a complete data analysis pipeline on the Titanic dataset, including data cleaning, feature engineering, and visualization.

---

## 📊 Project Overview
The goal of this project is to extract meaningful insights from the Titanic dataset using Python and data visualization techniques.

---

## 🛠️ Steps Performed

### 1. Data Cleaning
- Filled missing Age values using median
- Filled missing Embarked values using mode
- Dropped Cabin column due to excessive missing values

### 2. Feature Engineering
- Created Family Size feature (SibSp + Parch)
- Created Age Group categories (Child, Teen, Adult, Senior)

### 3. Data Visualization
- Histogram (Age Distribution)
- Bar Chart (Survival by Gender)
- Boxplot (Fare by Passenger Class)
- Scatter Plot (Age vs Fare with Survival)
- Heatmap (Correlation Matrix)

---

## 📈 Key Insights
- Females had a higher survival rate than males
- First-class passengers had higher fares and better survival chances
- Younger passengers were more common onboard
- Fare had a positive relationship with survival
